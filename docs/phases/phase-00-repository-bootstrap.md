# Phase 0: Repository Bootstrap

## Goal

Prepare the empty repository with documentation and a clear implementation path.

## Scope

Create:

- `AGENTS.md`
- `CODEX_HANDOFF.md`
- `README.md`
- `docs/MVP_SCOPE.md`
- `docs/MVP_PHASES.md`
- `docs/DECISIONS.md`
- `docs/API_CONTRACT.md`
- `docs/DOMAIN_MODEL.md`
- `docs/phases/`

## Non-Goals

Do not create backend code.

Do not create frontend code.

Do not add Docker logic.

Do not create database migrations.

## Deliverables

- Repository rules for future agents.
- MVP scope documentation.
- Phase plan.
- Decision log.
- Draft API contract.
- Draft domain model.
- Phase documentation directory.
- Updated handoff document.

## Verification

Phase 0 is verified when all requested documentation files and directories exist and no backend, frontend, or Docker implementation has been added.
