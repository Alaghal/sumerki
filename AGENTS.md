# Agent Instructions

## Language

Write progress updates in Russian. Final deliverables, repository documents, code, commit messages, and user-facing project text should be in English.

## Project Guardrails

Sumerki is in Phase 0: repository bootstrap.

Do not implement backend code, frontend code, Docker logic, database migrations, or runtime configuration during Phase 0.

Prefer small, phase-scoped changes. When a phase is requested, implement only that phase unless the user explicitly expands the scope.

## Documentation Rules

Keep project documentation current when changing scope, contracts, domain concepts, or implementation decisions.

Update `CODEX_HANDOFF.md` at the end of each work session with:

- current phase
- completed work
- important constraints
- next recommended step
- verification performed

## Repository Shape

Expected Phase 0 documentation files:

- `README.md`
- `CODEX_HANDOFF.md`
- `docs/MVP_SCOPE.md`
- `docs/MVP_PHASES.md`
- `docs/DECISIONS.md`
- `docs/API_CONTRACT.md`
